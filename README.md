# Setup

### Dependancies
- npm
- jq
- ssh
- curl
- adb
- [scrcpy](https://github.com/Genymobile/scrcpy)

### Install other dependancies
```bash
npm i
```

### Build
```bash
npm run-script build
```

# Run
```bash
./run.sh <username> <password> <path to SSH public key file (for whatever your default SSH key is)>
```
